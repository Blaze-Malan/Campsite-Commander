## Purpose of The App
The campsite commander app was created to help outdoor enthusiast to help oragnize and pack their gear before a trip

<img width="346" height="771" alt="Image" src="https://github.com/user-attachments/assets/674a40d0-5e5f-4a88-939a-a7fef1e0f7d9" />

## Design Considerations
- The app has a simple green and brown color grade to match the aesthetic of the outdoors
- Detects if the packing arrays are empty and displays a friendly user guide message rather than leaving an awkward blank screen.
- Utilizes a centralized DataManager object to share data across screens seamlessly without passing heavy, fragile intents.
